# ConvertX-CN 概覽

ConvertX-CN 是一個功能完整的自託管檔案轉換平台，支援 1000+ 種格式轉換。

---

## 什麼是 ConvertX-CN？

ConvertX-CN 是 fork 自 [C4illin/ConvertX](https://github.com/C4illin/ConvertX) 的增強版本，針對中文使用者與亞洲地區需求進行優化：

- **完整的轉換能力**：整合 20+ 種轉換引擎
- **中文優化**：內建 CJK 字型、OCR 語言包
- **一鍵部署**：Docker 容器化，無需安裝依賴
- **多語言介面**：支援 65 種語言
