## 如何使用

```bash
# 下載範例
curl -O https://raw.githubusercontent.com/pi-docket/ConvertX-CN/main/docs/Docker組合配置/compose.minimal.yml

# 重命名
mv compose.minimal.yml docker-compose.yml

# 建立 data 資料夾
mkdir -p data

# 修改 JWT_SECRET
nano docker-compose.yml

# 啟動
docker compose up -d
```
